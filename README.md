# vue-todo

> A Vue.js project

##### 简单的看板SPA 应用，使用了 vue vue-router vue-resource 等

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
